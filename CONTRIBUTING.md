# Contributing

Este repo está publicado como portafolio, pero PRs con mejoras son bienvenidos (siempre sin datos reales).

Ideas típicas:
- Exportar a Parquet
- Soporte de “resume” más fino (checkpoints por endpoint)
- Tests unitarios para normalización y dedup
- Conector a Postgres/BigQuery

Reglas:
- No subir `.env` ni tokens
- No subir CSVs reales
