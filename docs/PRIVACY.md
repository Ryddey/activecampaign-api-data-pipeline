# Privacidad y seguridad

Este proyecto está pensado para extraer datos potencialmente sensibles (contactos, correos, notas, tareas).

Recomendaciones mínimas:

- **Nunca** commitear `.env` ni tokens.
- No subir CSVs reales al repositorio público.
- Si compartes el proyecto como portafolio, usa data ficticia (ver `samples/`).
- Revisa términos y políticas de tu cuenta ActiveCampaign y de tu organización.
